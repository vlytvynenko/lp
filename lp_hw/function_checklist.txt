1. Did you start function definition with def?
2. Did you function name have only characters and _(underscore) characters?
3. Did you put an open parenthesis( right after function name?
4. Did you put your arguments right after parenthesis( separate by coma?
5. Did you make each argument unique (meaning no duplicated names)?
6. Did you put a close parenthesis and colon ): after arguments?
7. Did you indent all lines of code you want in this function four spaces? No more, no less!
8. Did you "end" you function by going back to writing with no indent (dedenting we call it)?

And when you run ("use" or "call") a function, check those things:

1. Did you call/use/run this function by typing it's name?
2. Did you put a parenthesis character ( after the name to run it?
3. Did you put the values you want into the parenthesis separate by coma?
4. Did you end the function call with ) character?
